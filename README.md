# 智教AI云 (EduGen AI Platform)

这是一个基于 Flask 的混合式教学平台原型系统，实现了从静态 UI 到动态 Web 应用的转换。系统集成了用户认证、角色管理（教师/学生）、数据看板、AI 助教模拟以及资源管理等功能。

## 功能特性

*   **用户认证**：注册、登录、注销、密码修改，支持 CSRF 保护。
*   **角色权限**：
    *   **学生**：访问课程主页、资源库、个人数据看板、AI 助教。
    *   **教师**：访问教师管理后台，查看学生行为风险评估，发布分层作业。
*   **数据驱动**：
    *   **Dashboard**：展示学生积分、排名、学习时长。
    *   **风险评估**：教师端根据学生积分自动标记风险等级（高/中/低）。
*   **AI 助教**：
    *   模拟对话接口，根据上下文提供教学建议。
    *   支持对接真实 OpenAI 接口（需配置 API Key）。

## 快速开始

### 1. 环境准备

确保您的系统已安装 Python 3.8+。

### 2. 安装与运行

**开发环境 (Development):**

```bash
# 给予脚本执行权限
chmod +x run_dev.sh

# 运行开发服务器 (会自动创建 venv 并安装依赖)
./run_dev.sh
```

**生产/部署环境 (Production):**

```bash
# 给予脚本执行权限
chmod +x deploy.sh

# 运行生产服务器 (Gunicorn)
# 首次运行会自动初始化数据库并写入测试数据
./deploy.sh
```

## 配置说明

项目配置位于 `config.py` 和 `.env` 文件（需自行创建）。

*   **数据库**：默认使用 SQLite (`app.db`)。
*   **AI 接口**：默认使用 Mock 模式。如需启用真实 AI，请设置环境变量：
    ```bash
    export AI_API_KEY="sk-..."
    ```

## 项目结构

*   `app/`：应用源码
    *   `models.py`：数据库模型
    *   `routes/`：路由控制器 (Main, Auth)
    *   `templates/`：Jinja2 页面模板
*   `assets/`：原始静态资源
*   `seed.py`：数据库填充脚本
